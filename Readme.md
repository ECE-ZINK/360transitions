## Repository for _Transitions of Quality Adaptation Mechanisms in 360° Video Streaming_ Paper

#### Structure
```
* 360player - C++ code for adaptive 360° video player 
* 360popularity - C++ code for cache initialization and popularity trace generation
* 360server - C++ code for HTTP server with throughput throttling
* eval - Gathered evaluation data and C++ code of simulations used
* preprocessing - Python script for tiling and segmenting equirectangular videos
```

Please refer to the readme files in each sub-folder for more details.
